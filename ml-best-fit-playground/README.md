\# ML Best Fit Playground



An interactive Streamlit app for exploring how model complexity affects generalization.



\## What it demonstrates



\- Underfitting vs overfitting

\- Model complexity vs performance

\- Impact of sample size

\- Effect of noise and dataset size



\## Features



\- Multiple true-function presets (linear, quadratic, sine, etc.)

\- Adjustable polynomial degree

\- Train vs validation RMSE metrics

\- Complexity sweep visualization across degrees



\## Run locally



```bash
pip install -r requirements.txt
streamlit run app.py
